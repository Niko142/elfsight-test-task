# Elfsight Test Task (Rick and Morty API)

## Описание

React-приложение для отображения персонажей из Rick and Morty API с пагинацией и фильтрацией.

## Ссылки

[Посмотреть live demo](https://niko142.github.io/elfsight-test-task/)

## Установка и запуск

1. Клонировать репозиторий

   ```bash
   git clone https://github.com/Niko142/elfsight-test-task.git
   cd elfsight-test-task
   ```

2. Установить зависимости

   ```bash
   npm install
   ```

3. Запустить проект в dev-режиме

   ```bash
   npm run start
   ```

   Приложение запустится по адресу: <http://localhost:3000>

4. Сборка production версии (опционально)

   ```bash
   npm run build
   ```

## Стек технологий

- React
- styled-components
- Axios
- customize-cra - настройка конфигурации `Create React App`
